# Mini-Projeto 07 — Lista de Funcionários

Mini-projeto desenvolvido para praticar o uso de Collections em Java,
com foco em organização de dados e manipulação de objetos.

O projeto consiste em criar funcionários, armazená-los em uma lista
e percorrer essa lista para exibir seus dados.

## Conceitos utilizados
- List
- ArrayList
- Programação Orientada a Objetos
- Encapsulamento
