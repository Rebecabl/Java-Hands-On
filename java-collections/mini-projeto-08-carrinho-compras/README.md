# Mini-Projeto 08 — Carrinho de Compras

Mini-projeto para praticar o uso de Collections em Java,
simulando um carrinho de compras com produtos e cálculo
do valor total.

## Funcionalidades
- Adicionar produtos ao carrinho
- Listar produtos
- Calcular total da compra

## Conceitos aplicados
- List e ArrayList
- Programação Orientada a Objetos
- Encapsulamento
- Regras de negócio simples
