# Java Hands-On

Este repositório reúne mini-projetos desenvolvidos ao longo do meu
processo de estudo e prática em Java.

Grande parte dos códigos já vinha sendo construída e testada
localmente no meu computador, durante meus estudos, e foi organizada
e versionada neste repositório de forma centralizada a partir deste
momento.

Os projetos estão estruturados de maneira progressiva, iniciando
pelos fundamentos da linguagem e evoluindo para conceitos como
Programação Orientada a Objetos, Collections, tratamento de erros,
persistência de dados e APIs REST.

O objetivo deste repositório é demonstrar tanto o domínio dos
conceitos quanto a evolução contínua da organização de código e das
boas práticas.
