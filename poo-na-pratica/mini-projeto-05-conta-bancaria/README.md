# Mini-Projeto 05 — Conta Bancária

Mini-projeto desenvolvido para praticar os conceitos básicos de
Programação Orientada a Objetos em Java, simulando uma conta bancária
com operações simples.

O sistema permite criar uma conta, realizar depósitos e saques,
respeitando regras básicas de negócio, como saldo disponível.

## Funcionalidades
- Criar conta bancária
- Depositar valores
- Realizar saques
- Consultar saldo

## Conceitos aplicados
- Classes e objetos
- Encapsulamento
- Métodos
- Validação de regras de negócio
