# Mini-Projeto 06 — Sistema de Funcionários

Este mini-projeto simula um sistema simples de funcionários,
utilizando Programação Orientada a Objetos em Java.

O objetivo é demonstrar a evolução do uso da linguagem Java,
saindo de fundamentos básicos para conceitos mais avançados
como herança e polimorfismo.

---

##  Problema Resolvido

Uma empresa possui diferentes tipos de funcionários, e cada
tipo possui uma forma diferente de calcular o salário.

- Funcionário CLT: salário fixo + benefício
- Funcionário PJ: pagamento por hora trabalhada

---

##  Estrutura do Projeto

- `Funcionario`  
  Classe abstrata que define os atributos e comportamentos comuns.

- `FuncionarioCLT`  
  Implementa o cálculo de salário para funcionários CLT.

- `FuncionarioPJ`  
  Implementa o cálculo de salário para funcionários PJ.

- `Main`  
  Classe principal responsável por instanciar os funcionários
  e demonstrar o uso do polimorfismo.

---

##  Conceitos Aplicados

- Classe abstrata
- Herança
- Polimorfismo
- Encapsulamento
- Organização em pacotes

---

## Objetivo Educacional

Demonstrar domínio de Programação Orientada a Objetos e
organização de código em Java, com foco em boas práticas
e clareza de implementação.
