# Mini-Projeto 09 — Validação de Dados

Mini-projeto desenvolvido para praticar tratamento de erros em Java
utilizando exceções personalizadas.

O sistema valida dados de um usuário e lança exceções quando regras
de negócio são violadas.

## Funcionalidades
- Criar usuário com validação de idade
- Lançar exceção personalizada
- Tratar erro com try/catch

## Conceitos aplicados
- Exceptions
- RuntimeException
- Encapsulamento
- Regras de negócio
