# Tratamento de Erros em Java

Este módulo reúne mini-projetos desenvolvidos para praticar
o tratamento de erros em Java, utilizando exceções para garantir
a integridade das regras de negócio.

Os exemplos abordam validações comuns em sistemas reais,
mostrando como lidar com dados inválidos de forma controlada
e segura.
